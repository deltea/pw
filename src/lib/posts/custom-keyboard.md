soooo i kinda fell into the custom mechanical keyboard rabbit hole again today...
i pretty much spent the entire day handcrafting my perfect keyboard, and it's a lot different from what my dream keyboard looked like a few months ago

first of all, the actual keyboard base. this keyboard i found on twitter is just so UGHGGHGH, it's the agar polycarbonate by kbdfans, with virtuoso keycaps:

![Agar PC Keyboard](https://pbs.twimg.com/media/G59E2xLbcAAYo0q?format=jpg&name=large)
<Caption>it just looks so beautiful i can't</Caption>

i LOVE the translucent polycarbonate material the base is made of, and also the curves make the board super unique.

for the switches, i wanted a more muted and creamy sound profile than what i currently have, so i would choose out of one of these:

- kkt kang white v3
- huano sakura v2
- hmx mango pudding

there's probably so many more (and probably better ones) but honestly idk if it really matters all THAT much. i am currently using stock brown switches so anything is probably better 😭

and now, for the part everyone's been waiting for, the keycaps! honestly i'm not completely sure what i would go with. the yakeylt keycaps has always been my dream keycap set (it looks so cute!) but i'm not sure if it would look good with the translucent base.

![Yakeylt Keycaps](https://unikeyboards.com/cdn/shop/files/liw4TBGB2P001gjT9Sjh8meZZYhm.jpg-large1400.jpg?v=1734713392&width=1100)
<Caption>bro these look so good like this might just be the best keycap set ever made</Caption>

but yeah that's it. i think i just want to build a custom keyboard that i will actually stick with. i might actually spend the summer grinding [stardance](https://stardance.hackclub.com/) for like $250 worth of keyboard grants, unless i already move onto another hyperfixation! (very likely)
